# Atividade 01 - MongoDB

Projeto desenvolvido como atividade prática de banco de dados não relacional utilizando MongoDB.

## Sobre o projeto
O projeto apresenta exemplos de criação de coleções, inserção de documentos e consultas com filtros, simulando cenários como biblioteca, escola e loja.

## Objetivo
Praticar os conceitos básicos de banco de dados NoSQL e operações fundamentais no MongoDB.

## Conteúdos abordados
- Criação de banco de dados
- Criação de coleções
- Inserção de documentos
- Consultas com filtros
- Estrutura de documentos no MongoDB

## Tecnologias utilizadas
- MongoDB
- Mongosh
